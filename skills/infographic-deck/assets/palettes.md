# Банк палитр

Каждый дек — одна палитра. Не использовать жёлто-бирюзовую из образца. Формат: ACCENT (акцент), ACCENT_D (тёмный оттенок акцента для второго тона), INK (текст), PANEL (светлая панель), MUTED (серый текст), LINE (хайрлайн). Фон всегда белый (FFFFFF), «выигрыш»/highlight — ACCENT.

Правила:
- Один ACCENT на всю деку, зафиксировать.
- Тёмные ударные слайды (титул, финал) можно делать на INK-фоне с ACCENT_D/светлым акцентом и кремовым текстом.
- Highlight-бокс: рамка ACCENT 2-2.5pt на белом, текст INK.
- Ротация: не повторять палитру предыдущего дека подряд.
- Проверить контраст: ACCENT на белом для крупного текста ок; мелкий текст только INK/MUTED.
